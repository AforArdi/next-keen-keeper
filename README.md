# Next Keen Keeper
### Live Link: https://next-keen-keeper.vercel.app/

A modern friend management application built to help you stay connected and maintain meaningful relationships. Track interactions, analyze your friendship patterns, and stay connected with the people who matter most.

## 📋 Description

KeenKeeper is a Next.js based web application that enables users to organize, track, and nurture their friendships effortlessly. With an intuitive interface, users can browse their friend directory, view detailed profiles, track communication history through a timeline, and gain insights into their social connections through interactive analytics. Whether you want to remember when you last contacted a friend or identify who might need a catch-up call, KeenKeeper has you covered.

## 🛠️ Technologies Used

- **Frontend Framework**: [Next.js](https://nextjs.org/) v16.2.3 - React framework with App Router
- **UI Library**: [React](https://react.dev/) v19.2.4
- **Styling**: [Tailwind CSS](https://tailwindcss.com/) v4 with [DaisyUI](https://daisyui.com/) components
- **Charts & Visualization**: [Recharts](https://recharts.org/) v3.8.1
- **Icons**: [React Icons](https://react-icons.github.io/react-icons/) v5.6.0
- **Notifications**: [React Toastify](https://fkhadra.github.io/react-toastify/introduction) v11.0.5
- **Loading States**: [React Spinners](https://www.react-spinners.com/) v0.17.0
- **Performance**: React Compiler (Babel plugin) for automatic optimization
- **Code Quality**: ESLint v9 for code linting

## ✨ Key Features

### 1. **Friend Directory**
Browse all your friends in a responsive, visually appealing grid layout. Each friend card displays their profile picture, name, and connection status. Quickly navigate to detailed profiles to view comprehensive information about each friend, including their contact information, tags, and relationship goals.

### 2. **Friendship Analytics Dashboard**
Visualize your social connections with interactive pie charts that provide insights into your friendship patterns.

### 3. **Activity Timeline**
Track all interactions with your friends through a comprehensive timeline. Filter activities by type—Calls, Text messages, or Video calls—to see exactly how you're staying in touch.
